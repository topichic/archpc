# Лабораторная работа №5

**Инструменты:** PassMark MonitorTest v4, монитор HP X24c

---

## Ход выполнения работы

Характеристики определены с помощью утилиты MonitorTest и данных EDID монитора.

---

### 2. Выполнение тестов в MonitorTest

Настройки программы перед началом тестирования:
- **Разрешение:** 1920 x 1080 - 32bpp - 144 Hertz
- **Режим:** Тестирование одного монитора

#### Тест 1: Scale (Масштабирование / Битые пиксели)

**Цель:** Проверка равномерности подсветки и выявление битых пикселей. Яркость и цвет на всем экране должны быть одинаковыми.

| Тест | Результат | Примечание |
| :--- | :--- | :--- |
| Scale (Black - White) | **Успешно** | Белое поле равномерное, засветов по краям не обнаружено. |
| Scale (Black - Red) | **Успешно** | Красный цвет насыщенный, без темных пятен. |
| Scale (Black - Green) | **Успешно** | Зеленый цвет равномерный. |
| Scale (Black - Blue) | **Успешно** | Синий цвет равномерный. |

**Вывод:** Битые пиксели и дефекты подсветки не обнаружены. Монитор исправен.

---

#### Тест 2: Gamma (Гамма-кривая)

**Цель:** Проверка плавности цветовых переходов от черного к цветному изображению.

| График | Оценка |
| :--- | :--- |
| Gamma (White) | Кривая плавная, без резких скачков. Градиент от черного к белому отображается корректно. |
| Gamma (Red) | Плавный переход от черного к красному. Детали в темных областях различимы. |
| Gamma (Green) | Переход плавный, без искажений. |
| Gamma (Blue) | Переход плавный, без искажений. |

**Вывод:** Гамма-коррекция монитора (значение 2.20 из EDID) настроена правильно, цветопередача соответствует стандарту sRGB.

---

#### Тест 3: Contrast (Контрастность)

**Цель:** Определение минимального шага яркости, при котором оттенки остаются различимыми. Оптимальным считается шаг в 20%.

| Тест | Процент различимости | Оценка |
| :--- | :--- | :--- |
| Contrast (Red) | ~15% | Отличная детализация в красных тонах. |
| Contrast (Green) | ~20% | Хорошая детализация. |
| Contrast (Blue) | ~20% | Хорошая детализация. |
| Contrast (White) | ~10% | Очень высокая чувствительность к градациям белого. |

**Вывод:** Контрастность монитора находится на высоком уровне. Все цвета хорошо различимы, что говорит о качественной матрице.

---

#### Тест 4: LCD Pixel Persistence (Инерционность пикселей)

**Цель:** Оценка скорости отклика матрицы. Чем короче шлейф от движущегося объекта, тем лучше.

| Параметр | Результат |
| :--- | :--- |
| Видимый шлейф | Короткий, едва заметный. |
| Оценка | **Отлично** |

**Вывод:** Благодаря частоте обновления 144 Гц, матрица обладает низкой инерционностью. Монитор отлично подходит для динамичных сцен (игры, видео).

---

#### Тест 5: Printer Test (Сравнение с печатными цветами)

**Цель:** Выявление областей, где цвета на экране сливаются в белый и не различимы.

**Результат:** В диапазоне ячеек с очень светлыми пастельными оттенками (например, `A1-D4`) цвета становятся практически неразличимы на белом фоне. Это связано с ограничением цветового охвата монитора по сравнению с печатной машиной.

---

### 3. Общий вывод по работе

В ходе выполнения лабораторной работы был проведен комплексный анализ монитора HP X24c с использованием утилиты PassMark MonitorTest.

1.  **Качество матрицы:** Монитор не имеет битых пикселей и дефектов подсветки.
2.  **Цветопередача:** Гамма-кривая и цветовой баланс настроены корректно (гамма 2.20). Контрастность позволяет различать большое количество оттенков.
3.  **Производительность:** Высокая частота обновления (144 Гц) обеспечивает минимальную инерционность пикселей, что критично для динамичных задач.

**Итог:** Монитор HP X24c является высококачественным устройством, подходящим как для профессиональной работы с графикой и цветом, так и для динамичных игр.

## Контрольные вопросы

### 1. Виды мониторов.

`По типу используемой технологии мониторы делятся на: ЖК (LCD) с подтипами TN, IPS и VA; органические светодиодные (OLED), а также устаревшие плазменные (PDP) и электронно-лучевые (ЭЛТ).`

### 2. Конструкция ЖК-мониторов.

`Основные компоненты: источник подсветки (LED), светорассеивающие пленки и поляризационные фильтры, а также жидкокристаллическая матрица, состоящая из пикселей, управляемых тонкопленочными транзисторами (TFT).`
